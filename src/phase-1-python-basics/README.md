## 📚 核心概念

### NumPy

NumPy 是科学计算的基础库，提供：

- N 维数组对象 (ndarray)
- 广播机制 (Broadcasting)
- 线性代数、傅里叶变换等数学函数

### Pandas

Pandas 是数据分析的核心工具：

- DataFrame 和 Series 数据结构
- 数据清洗和预处理
- 数据聚合和分组操作

### Matplotlib

Python 最基础的可视化库：

- 折线图、散点图、柱状图
- 多子图布局
- 自定义样式
