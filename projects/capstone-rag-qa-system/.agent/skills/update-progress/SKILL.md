# 更新开发进度技能

此技能用于更新 DocuMind AI 项目的开发进度追踪。

## 进度文件位置

```
projects/capstone-rag-qa-system/docs/PROGRESS_TRACKER.md
```

## 状态标记

| 图标 | 状态   | 说明     |
| ---- | ------ | -------- |
| ⬜   | 待开始 | 尚未开始 |
| 🔄   | 进行中 | 正在开发 |
| ✅   | 已完成 | 开发完成 |
| ⚠️   | 阻塞   | 遇到问题 |
| ❌   | 取消   | 任务取消 |

## 更新任务状态

### 开始任务

将状态从 `⬜ 待开始` 改为 `🔄 进行中`：

```markdown
| 2.1 | 实现 BaseParser 抽象类 | 0.5h | 🔄 进行中 | | |
```

### 完成任务

将状态改为 `✅ 已完成`，并填写完成日期：

```markdown
| 2.1 | 实现 BaseParser 抽象类 | 0.5h | ✅ 已完成 | 2026-01-27 | 完成 |
```

## 阶段完成检查

每个阶段完成时：

1. ✅ 所有任务标记为完成
2. ✅ 所有交付物已勾选
3. ✅ 进度条更新为 100%
4. ✅ 统计表数据更新
5. ✅ 在每日日志记录完成
6. ✅ 更新总体进度
