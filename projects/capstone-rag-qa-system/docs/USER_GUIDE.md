## 快速开始

### 访问系统

在浏览器中打开：

- 本地部署: `http://localhost:5173`
- Docker 部署: `http://localhost`

### 界面概览

系统主界面包含：

- **侧边栏**: 导航菜单，可切换知识库列表、对话、设置
- **主区域**: 显示当前页面内容

## 知识库管理

### 创建知识库

1. 点击首页右上角 **「创建知识库」** 按钮
2. 输入知识库名称（必填）
3. 输入描述（可选）
4. 点击 **「创建」** 完成

### 查看知识库

- 首页以卡片形式展示所有知识库
- 每个卡片显示：名称、描述、文档数量、分块数量、更新时间

### 删除知识库

1. 将鼠标悬停在知识库卡片上
2. 点击右上角 **「⋯」** 菜单按钮
3. 选择 **「删除」**
4. 确认删除

> ⚠️ **警告**: 删除知识库将同时删除所有相关文档和对话记录

## 文档上传

### 支持的格式

| 格式     | 扩展名 | 说明                |
| -------- | ------ | ------------------- |
| PDF      | .pdf   | 支持文本 PDF        |
| Word     | .docx  | Microsoft Word 文档 |
| 文本     | .txt   | 纯文本文件          |
| Markdown | .md    | Markdown 格式       |

### 上传步骤

1. 进入知识库的文档管理页面
2. 点击 **「上传文档」** 按钮
3. 拖拽文件到上传区域，或点击选择文件
4. 等待上传和处理完成

### 处理状态

| 状态   | 说明                     |
| ------ | ------------------------ |
| 等待中 | 文件已上传，等待处理     |
| 解析中 | 正在提取文档内容         |
| 分块中 | 正在将文档切分为语义片段 |
| 向量化 | 正在生成向量嵌入         |
| 已完成 | 处理完成，可以问答       |
| 失败   | 处理失败，请检查文件     |

## 智能问答

### 开始对话

1. 在知识库页面点击 **「开始对话」**
2. 在底部输入框输入问题
3. 按 Enter 或点击发送按钮

### 对话功能

- **流式输出**: 答案逐字显示，无需等待完整响应
- **来源引用**: 每个回答附带相关文档片段来源
- **对话历史**: 点击"历史"按钮查看过往对话
- **新对话**: 点击"新对话"开始新轮对话

### 反馈功能

每个 AI 回答下方有 👍 👎 按钮，可以提交反馈帮助改进系统。

## 常见问题

### Q: 文档处理失败怎么办？

A: 请检查：

1. 文件格式是否支持
2. 文件是否损坏
3. PDF 是否为扫描件（当前不支持 OCR）

### Q: 回答不准确怎么办？

A: 尝试：

1. 使用更具体的问题
2. 确保相关文档已上传并处理完成
3. 检查问题是否与文档内容相关

### Q: 如何提高问答质量？

A: 建议：

1. 上传高质量、格式规范的文档
2. 确保文档内容与查询主题相关
3. 使用清晰、具体的问题表述
